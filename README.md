 AP-Chat-app-notepad-poker-game-
This repository contains three Java projects in one place:

- Chat App
- Notepad App
- Poker Game

These projects were created using Java to practice desktop application development, user interaction, object-oriented programming, and basic game logic.

 Projects

 1. Chat App

The Chat App is a simple Java messaging application. It provides a basic chat-style interface where users can type and view messages.

 Features

- Send messages
- View messages in a chat window
- Simple Java GUI
- Basic message handling

  2. Notepad App

The Notepad App is a Java-based text editor application. It allows users to write, edit, and manage text in a simple notepad interface.

 Features

- Simple notepad interface
- Write and edit text
- Manage text content
- Java-based application structure

 3. Poker Game

The Poker Game is a Java-based card game application. It allows users to play a simple five-card draw poker game against a dealer.

 Features

- Simple poker game interface
- Place bets
- Replace selected cards
- Display poker hand results
- Java-based application structure
